中科院信工所项目
本系统是基于可信度算法的多模型协同检测系统，可以实现在线实时检测恶意域名

app——基于flask框架搭建服务器

dga_detection——实现了多模型的协同检测

feature_extraction——用于特征的提取

dname_add——添加域名数据的脚本文件

dname_query——查询域名数据的脚本文件

dname_del——删除域名数据的脚本文件

requirement——介绍了所需环境，包的版本

本系统在python3.8的环境进行编写和运行，具体见requirements.txt

部署好环境之后，只需要运行app.py即可，通过浏览器可以访问。
